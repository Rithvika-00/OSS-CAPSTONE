Project Description



This repository presents my capstone project for the Open Source Software course. The project focuses on an in-depth audit of Git, examining its background, licensing model, and importance in today’s open-source ecosystem.



The aim is to understand how Git supports collaboration, transparency, and distributed development in modern software engineering.



Project Components



The project is divided into two main sections:



1\. Audit Documentation

A comprehensive report (PDF) that explores:



The history and development of Git

Its licensing under GPL

Its widespread usage in open-source and industry

Ethical considerations in open collaboration



2\. Shell Script Implementation

A set of five Bash scripts demonstrating practical system interaction and automation tasks in Linux.



Repository Contents

OSSCapstoneProject.pdf → Complete audit report (12–16 pages)

script1.sh → Displays system identity information

script2.sh → Verifies Git installation and explains its philosophy

script3.sh → Automates backup of selected files

script4.sh → Analyzes system logs based on patterns

script5.sh → Creative script highlighting open-source freedom



Execution Instructions



All scripts are intended to run in a Linux environment.



Use the following commands to execute any script:

chmod +x script\_name.sh

./script\_name.sh
